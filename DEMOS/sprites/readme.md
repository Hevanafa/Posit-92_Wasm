# Sprites Demo

This demo project demonstrates a few bitmap blitting features that are included in **Posit-92**

- Full sprite: `spr`
- Sprite region: `sprRegion`
- Full sprite with alpha blending: `sprBlend`
- Sprite scaling: `sprStretch` or `sprScale`
- Region scaling: (not yet implemented)
- Sprite flipping: `sprFlip`
- Sprite rotation: `sprRotate`

The external assets are as follows:
- The blue enemy of Desert Shooter Pack from Kenney.nl
- Slime girl by piyo_0426
- Cursors from Windows XP
